ARQUITECTURA CRÍTICA

ARQUITECTURA DE LA COLMENA

Inventario técnico del núcleo de proceso, la red y los puestos de mando que sostienen la operación.

NÚCLEO

NÚCLEO DE PROCESO

Cluster de virtualización sobre Dell PowerEdge R650 con Proxmox VE para orquestar sistemas Windows Server y Ubuntu Server enfocados a directorio, servicios y datos.

RED

SISTEMA CIRCULATORIO

Conmutación sobre Cisco Catalyst, VLANs segmentadas y un perímetro controlado para aislar tráfico, administrar accesos y reducir la superficie de exposición.

SOC

PUESTOS DE MANDO

Estaciones de trabajo preparadas para análisis forense, respuesta a incidentes y administración segura de la infraestructura.

SECTORES PRINCIPALES

El sistema distribuye los recursos por dominios funcionales para mantener aislamiento, observabilidad y continuidad.

S1 · NÚCLEO DE PROCESO

Virtualización, directorio, servicios y custodia de datos.

S2 · SISTEMA CIRCULATORIO

Switching, seguridad perimetral y conectividad inalámbrica.

S3 · PUESTOS DE MANDO

Análisis SOC, investigación y forense digital.

S4 / S5 · BACKUP Y ENERGÍA

Persistencia, recuperación y soporte vital del sistema.
